# Probitas Client

Client library collection for Probitas scenario testing framework.

## Quick Reference

- **Runtime**: Deno 2.x
- **Registry**: JSR (`@probitas/*`)
- **Reference Project**:
  [jsr-probitas/probitas](https://github.com/jsr-probitas/probitas) (main
  Probitas framework)

## Commands

```bash
deno task verify      # Run format, lint, type check and tests (USE THIS)
deno task test        # Run tests only
deno task check       # Type check only
```
